# Calculator

Schreib eine Taschenrechner-App die folgende Operationen ausführen kann:

- Addition
- Subtraktion
- Multiplikation
- Division
- Modulo

1. Schreib eine Funktion, die zwei Zahlen je nach Operation berechnet.

1. **Optional:** Du kannst Bootstrap benutzen um die App zu stylen.
